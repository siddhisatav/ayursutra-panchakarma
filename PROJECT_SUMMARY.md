# 📋 Project Summary - AyurSutra Panchakarma

## 🎯 Project Overview

**AyurSutra Panchakarma** is a comprehensive, production-ready static frontend website for an Ayurvedic Panchakarma therapy center. Built with modern web technologies, it features role-based dashboards, appointment management, and therapy tracking.

## 🔐 User Roles & Permissions

### Patient
- View personal dashboard
- Book appointments
- Track therapy progress
- Read Panchakarma guide
- Add personal notes
- View diet plans

### Therapist
- View assigned patients
- Manage appointments (accept/reject)
- Add therapy notes
- Assign diet plans
- View daily schedule
- Update profile

### Practitioner
- View center overview
- Manage therapists
- Monitor all patients
- Configure settings
- View statistics
- Manage schedules

## 💾 Data Architecture

### localStorage Keys
```
ayur_current_user          - User session
ayur_appointments          - All appointments
ayur_notes                 - User notes
ayur_diet_plans           - Assigned diet plans
ayur_therapist_availability - Therapist schedules
```
